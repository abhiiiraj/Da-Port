# Amazon Sales Dashboard (2019–2021, USA)

This repository contains a Power BI dashboard and supporting data based on cleaned Amazon sales data for the USA between 2019 and 2021.

## 📊 Dashboard Overview

The Power BI report (`Amazon Dashboard 2019-2021 USA.pbix`) includes insights into:

- Total sales and revenue
- Category-wise performance
- Monthly/Yearly trends
- Top-performing products
- Customer behavior

## 📁 Files

- `Amazon Dashboard 2019-2021 USA.pbix`: The main Power BI dashboard file
- `Cleaned_Amazon_Store_Sales_Data.csv`: Cleaned sales data used in the dashboard

## 🛠️ How to Use

1. Clone or download the repo
2. Open the `.pbix` file using [Power BI Desktop](https://powerbi.microsoft.com/desktop/)
3. Explore or extend the report

## 📌 Data Source

This dataset has been cleaned and prepared from Amazon store sales data covering the period 2019–2021 for the USA.
